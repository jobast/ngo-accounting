CREATES - Documents
====================

Ce dossier contient les documents a telecharger.

Fichiers requis:
- Manuel_Gestion_CREATES.pdf : Manuel de gestion administratif et financier

Instructions:
1. Convertir le fichier "Manuel de Gestion CREATES.docx" en PDF
2. Renommer le fichier en "Manuel_Gestion_CREATES.pdf"
3. Placer le fichier dans ce dossier (static/docs/)

Le fichier sera accessible via le bouton "Telecharger PDF"
dans la section Aide > Manuel de gestion.
